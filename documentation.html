Welcome to the Veteran API!





DOCUMENTATION FOR THE VETERAN APP API

BELOW ARE THE FOUR TABLES AND THE URLS THAT REQUESTS NEED TO BE SENT TO IN ORDER TO INTERACT WITH THE DATABASE:

TO POST INFO, YOU NEED TO SEND AN OBJECT WITH KEY-VALUE PAIRS IN THE "DATA" FIELD OF THE JSON OBJECT YOU SEND!!!!
----------------------------------------------------------------------------------------------

USERS API:

TO GET ALL USERS
GET https://vet-codeathon.herokuapp.com/users/

------------------------------------------------------------------------------------------------

TO GET A USER BY ID
GET https://vet-codeathon.herokuapp.com/users/{id}

---------------------------------------------------------------------------------------------------

TO MAKE A NEW USER
POST https://vet-codeathon.herokuapp.com/users/new

^^^NEEDS THESE ATTRIBUTES IN BODY:
{
    firstName:
    lastName:
    email: !!!THIS IS UNIQUE, ONE ACCOUNT PER EMAIL, THE DATABASE REJECT DUPLICATES!!!
    password:
    imageURL <OPTIONAL>:
}

-------------------------------------------------------------------------------

TO UPDATE A USER
PUT https://vet-codeathon.herokuapp.com/users/{id}

^^^NEEDS THESE ATTRIBUTES IN BODY:
{
    firstName:
    lastName:
    email : !!!THIS IS UNIQUE, ONE ACCOUNT PER EMAIL, THE DATABASE REJECT DUPLICATES!!!
    password:
    imageURL <OPTIONAL>:
}

----------------------------------------------------------

TO DELETE A USER
DELETE https://vet-codeathon.herokuapp.com/users/{id}
